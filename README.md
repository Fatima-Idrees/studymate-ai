# StudyMate AI - AI Powered Study Assistant

## Overview

StudyMate AI is a full-stack study assistant that helps students learn more efficiently using AI. The application allows users to generate notes, quizzes, flashcards, and ask study-related questions through Google's Gemini AI. It also provides secure authentication and tracks user progress with a clean, responsive interface.

## Features

* User registration and secure login using JWT authentication
* AI-powered note summarization using Google Gemini API
* Generate quizzes from study content
* Create AI-generated flashcards for quick revision
* Interactive AI Q&A assistant for study-related questions
* Personal dashboard to manage notes and track learning progress
* Responsive design for desktop and mobile devices
* Store user data and study materials in MongoDB

## Technologies Used

* React
* Vite
* Node.js
* Express.js
* MongoDB
* Google Gemini API
* JWT Authentication
* React Router DOM
